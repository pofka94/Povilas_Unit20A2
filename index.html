What is Javascript and what are its uses?

Javascript, also abbreviated as JS, is one of the core programming languages that is used in web development.
As it can achieve many things, such as 2D/3D animation, video players, interactive maps, etc.

According to Mozilla, JavaScript runs on client web browsers, which can be used to design or program how the web pages behave on the occurence of an event.
JavaScript is also described as an easy to learn and powerful scripting language, that is because it's syntax is similar to C++ and Java, as it reduces the number of new concepts required to learn.

What are the major uses of JavaScript?

There are many, but these are just a few:

  • Creating web and mobile apps
  • Building web servers and developing server applications
  • Game development
  • Adding interactivity

Creating web and mobile apps

Web and/or mobile app developers can use JavaScript frameworks for developing and making web and mobile apps.
JavaScript frameworks are a collection of JavaScript code libraries that provide developers with pre-written code to use for routine programming features and tasks.

An example of popular JavaScript frameworks would be React, React Native, Angular and Vue.

Building web servers and developing server applications

Allows the creation of simple web servers and develop the back-end infrastructure using Node.js.

Game development

Developers can create games using pure JavaScript that can work on browsers.

Webpage interaction

Allows to make all kinds of things, whether it is an animation, a game, using drop-down menus, etc.

What are the limitations of JavaScript?

  • Client-side Security - Javascript is executed on the client-side, and because of that, JavaScript does not allow the reading or writing of files.
  • No multiprocessor/multi threading capabilities - JavaScript would run a lot quicker if it had these capabilities but as of right now, JavaScript uses only a single-thread of a CPU.
  • No support for network applications.
  • Cannot access databases and would need to rely on a server side language.
  • Browser support - Some browsers can give different results when interacting with JavaScript.
  • Not search engine friendly.

How to add JavaScript to HTML:

There are 3 ways of adding JavaScript to webpages:

  1. Embedding code
  2. Inline code
  3. External file

Embedding code:

There are 2 ways of embedding JavaScript to HTML pages, and both use the "<script>...</script>" tag.
Javascript can either be defined in the "<head>" tag or in the "<body>" tag.

Example1:

   
            <!DOCTYPE html>
            <html>
                <head>
                    <title>Embedding example 1</title>
                    <script>
                    function embedEx1(){ 
                        alert("Exmaple 1") 
                    };
                    </script>
                </head>
                <body>
                    <p>I will show you the world!</p>
                </body>
            </html>


Example2:

   
            <!DOCTYPE html>
            <html>
                <head>
                    <title>Embedding example 2</title>
                </head>
                <body>
                    <script>
                    function embedEx2(){ 
                        alert("Example 2") 
                    };
                    </script>
                    <p>I will show you the world!</p>
                </body>
            </html>


Inline code:

Usually, this method is used with HTML event attributes, such as onmouseover, onclick, onerror and many more.
For most of my buttons I used the onclick event, as it lets me to call out a function.

Inline example:

   
            <!DOCTYPE html >  
            <html>  
                <head>  
                    <title>Inline example</title>  
                </head>  
                <body>  
                // Should pop up an alert window saying "Thank you for clicking on me!"
                    <button onclick="alert('Thank you for clicking on me!')">Click me!</button> 
                    <p>Hello there!</p>
                </body>  
            </html>

External file:

A separate file can be made to hold all of the JavaScript code.
This makes the raw HTML look cleaner while also allowing other webpages to reuse the JavaScript code.

Usually, the external(.js) files are put at the bottom of the website, that is because when you place your JavaScript links at the bottom of your HTML body,
it gives the HTML time to load before any of the JavaScript loads, which can prevent errors, and speed up website response time.

External file example:

   
            <!DOCTYPE html >  
            <html>  
                <head>  
                    <title>External file example</title>  
                </head>  
                <body>
                    <p>Why did the duck cross the road?<br>
                    I don't know...</p>
                    <script src="scripts/js.js"></script> // defines the path to the location of the .js file and grabs it.
                </body>  
            </html>

Variables:

A JavaScript variable hold data values with a name assigned to them to be used later on.
A variable in JavaScript does not need any data to exist, is just needs to be defined.

There are 3 keywords that can define a variable:

  1. var
  2. let
  3. const

According to mozilla, "let" allows you to declare variables that are limited to the scope of a block statement, or expression on which it is used.
While the "var" keyword, declares a variable grobally, or locally to an entire function.

In simple term, what it means is that if the variable that uses "var" keyword inside of a function, then it can still be accessed inside a different block,
while "let" would only work inside the current block(such as a function).

Here is an example or var and let:

   
            const text = "The result is: "  // A string variable that cannot be reassigned.
            let summary;    // A real variable with the data set as undefined
            var x = 14;     // an assigned variable with the value of 14.
            var y = 7;
        
            // assigns the variable "summary" the data of "text" and the result of x + y. Then shows what the variable "summary" is.
            // Should show "The result is: 21"
            summary = text + (x + y)
            alert(summary);

The keyword "const" are block-scoped in the same way as "let" except "const" cannot be reassigned or redeclared.

An example of const:

   
            const x = 40;   // sets a constant variable with a value of 40

            function constEx(){
                try { x = 21;   // tries to set the value of x to 21
                } catch (something) { alert(something) }    // catches the result and then shows it
                // should show an error message
            
                alert("The number is: " + x)    // after the error message it should show the text "The number is: 40"
            }

Loops:

Loops allows to repeat the same function or line of code multiple times without the need of repeating it yourself.
In this example I will be using a "for" loop.

The way a "for" loop works is by defining a variable, then using it to set a condition, then increments it once it finishes one loop and breaks until the condition is no longer true.

For example:

   
            for (let i = 0; i < 3; i++) { // shows 0, then 1, then 2, then breaks
                alert(i);
            }

Decision Making

Decision making allows the code to have more than 1 possible returns.

For this example, I will use an "if" statement.
An "if" statement uses some kind of a parameter with a condition to decide what it wants to do.
It can also put in other conditions, if the first one does not come true.

Example:

   
            var x = 10;

            if(x >= 18){    // if the number x is more or equal to 18 then run this path, if not then skip.
                return alert("Accepted!")
            } else if (x >= 15) {   // if the number x is more or equal to 15 then run this path, if not then skip.
                return alert("So close, but not there!")
            } else {    // if nothing returns true, then do this.
                return alert("Too young!")
            }
